
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. Players will guess letters to reveal a hidden word before running out of attempts. This assignment practices string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Create a complete Hangman game that allows players to guess letters and try to figure out a randomly selected word within a limited number of attempts.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in the format `_ _ _`
- Track incorrect guesses remaining and display them to the player
- End the game when the word is guessed or attempts are exhausted
- Display clear win/lose messages at the end of the game
